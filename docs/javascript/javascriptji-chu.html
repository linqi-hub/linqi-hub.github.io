    # 函数的定义

    函数和对象功能都是一样的，唯一不同的是函数可以调用。

    1. 函数声明

      // 下面这几个元素一个都不能少
function myName(){}
      1
      2

      函数也可以定义在另一个函数之中

    2. 函数表达式

      var myName = function(){}
      1

      也可以作为其他函数调用的参数或返回值:

      myFunc(function(){})
      1

      立即执行函数：

      (function(){})()
      1

      创建一个函数后，立即执行。需要把创建的函数用小括号给包裹起来，这样可以让JS解析器知道这个立即执行函数是一个函数表达式，而不是一个函数声明语句。

    3. 构造函数

      当函数通过new进行调用时，这个函数就是构造函数，构造函数的首字母要大写。

    4. 箭头函数

      箭头函数就是函数表达式的一种简化版，更加简洁明了。

      (参数) => {函数体}
      1

    5. 生成器函数

    # 函数的参数

    • 形参：定义函数时所列举的变量

      function myName(a, b){}
      1

    • 实参：调用函数时传递给函数的值

      myName(a, b)
      1

    如果实参的数量大于形参，多的实参不会赋值；

    如果形参的数量大于实参，多的形参会赋值为undefined；

    • 剩余参数（ES6）

      剩余参数是一个数组，用...表示：

      function myName(first, ...arguments)
      1

    • 默认参数（ES6）

      默认参数适用于每次函数调用都会用同样参数的场景。

      // 创建默认参数的方法是为函数的形参赋值
function myName(a, b = 'hello'){}
      1
      2

    • 隐含参数

      每个函数被调用的时候都会传递两个隐式参数this和arguments。

      • arguments表示传递给函数的所有参数的集合，通过length属性可以获取实参的个数。
      • this表示被调用函数的上下文对象。

    # 执行上下文

    在js中我们时刻使用函数，使用函数计算、更新UI、复用代码，当函数调用函数，并且调用完成时，程序会回到函数调用的位置，那么js是如何跟踪函数的执行？--通过执行上下文来跟踪代码。

    执行上下文分为两种：

    • 全局执行上下文：只有一个，在js代码执行的时候就已经确立；
    • 函数执行上下文：每次调用函数的时候，就会创建一个函数执行上下文；

    # 函数的调用

    # 1.函数的直接调用

    通过（）直接调用函数，并且函数表达式不是作为对象的属性存在，仅仅是一个函数，这时候就是直接调用函数。

    这时候调用时this的值就是全局上下文（window对象）。

    function myName(){}
myName()

    # 2.作为方法调用

    函数作为另一个对象的属性，并且通过对象属性引用的方式调用函数时，函数会作为对象的方法被调用，这时候的函数就称为方法。

    var myObj = {}
// 把函数赋值给对象的属性
myObj.handsome = function(){}
// 通过对象的方法调用
myObj.handsome()

    # 3.作为构造函数调用

    1. 形式

      函数作为构造函数去调用就是普通函数使用new进行调用，通常以大写字母开头。

      构造函数的目的是创建一个新对象，并进行初始化设置，然后将这个对象作为构造函数的返回值

    // 创建函数
function myName(){}
// 作为构造函数去调用
new myName()

    3
    4

    这时候this是新构造的对象。

    1. 原理

      当使用new去调用构造函数的时候会自动执行一下操作：

      • 创建一个空的对象；
      • 这个空对象被设置为该函数的上下文；
      • 新构造的对象作为new运算符的返回值；

    var Func = function Func(){}
var func = new Func()
// 创建一个全新的对象
var obj = new Object()

// 设置原型链
obj.__proto__ = Func.prototype

// 设置函数上下文
var result = Func.call(obj)

// 返回新构造的对象。
// 如果是值类型，直接返回。
// 如果是引用类型，则返回这个引用类型的对象。
if(typeof result == 'object'){
  func = result
}else{
  func = obj
}

    # 4.通过函数apply或者call方法调用

    如果想要改变函数的上下文，可以使用call或者apply进行调用。

    每个函数上面都存在这两种方法：call和apply。

    // 创建函数
function myName(){}
// 创建对象
var obj = {}
// 使用call绑定函数的上下文为obj
foo.call(obj)
    1

    3
    4

    6

    这时候的this就是绑定的对象。

    传入call和apply方法的第一个参数会被作为函数上下文，他俩的不同在于后续的参数，apply只需要一个包含参数的数组，而call需要传递任意数量的参数值。

    如果第一个参数传入的对象调用者是null或者undefined，call方法将把全局对象（浏览器上是window对象）作为this的值。
